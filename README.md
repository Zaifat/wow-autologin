# Менеджер персонажей WOW 3.3.5a
<img width="877" height="587" alt="image" src="https://github.com/user-attachments/assets/10ca3cc6-89db-4dbe-957f-14b3b9b986f9" />

Лаунчер для World of Warcraft 3.3.5a (WotLK) с автологином.
Один клик по карточке персонажа — клиент сам вводит логин, пароль, выбирает реалм, заходит за нужного персонажа и (если подключено) подтверждает 2FA-код.

## Возможности

- **Установщик** — ставится в `Program Files\WowManager` (или per-user под AppData), добавляет ярлык в Пуск, опциональный ярлык на рабочем столе и автозапуск при старте Windows
- **Записи только с аккаунтом** — можно добавить аккаунт без персонажа; аккаунты и персонажи показываются отдельными блоками
- **Ярлыки на запись** — кнопка в настройках создаёт на рабочем столе ярлык, запускающий конкретного персонажа/аккаунт одним кликом (зависит только от самой программы)
- **Drag-and-drop** — порядок записей меняется перетаскиванием мышью
- **Авто-бэкап WTF и аддонов** — перед запуском создаётся .zip-снимок папки `WTF` и `Interface/AddOns`. Настраивается число копий, интервал и папка (с запросом прав администратора, если папка их требует); есть восстановление из любого снимка
- **Системный трей** — закрытие окна сворачивает в трей, оттуда быстрый запуск первых 15 персонажей; полный выход через пункт «Выход»
- **Single-instance** — повторный запуск exe просто разворачивает уже открытое окно
- **Тёмная и светлая темы** + **английская локализация** — переключение в Настройки → Доп настройки. Цвета классов в таблице адаптируются под тему
- **2FA TOTP** — Google Authenticator / 2FAS Auth / Yandex.Ключ (RFC 6238). Код генерируется на лету и автоматически отправляется в окно подтверждения
- **Несколько аккаунтов** — каждый персонаж со своим логином/паролем/реалмом/realmlist
- **Шифрование DPAPI** — пароли и секреты 2FA в `characters.json` шифруются Windows-ключом профиля; можно включить или выключить
- **Подсветка по классам** (класс необязателен), поиск, поле **ГС** с автоподстановкой из SavedVariables (если установлен аддон GearScore)
- **Сортировка по любому столбцу** клик по заголовку (повторный — обратный порядок), ГС сортируется численно
- **Бэкап/перенос конфига** — кнопки «Загрузить» / «Скачать» / «Очистить» в настройках
- Имя персонажа поддерживает **кириллицу** (исправлен баг с шагом массива в оригинальном AwesomeWotlk — `CharVectorEntry = 0x198` байт)

## Использование

1. Скачать `Manager_WOW.exe` из [Releases](../../releases) — это установщик
2. Установить (по умолчанию в `Program Files\WowManager`, можно выбрать другую папку) → запустить
3. «Настройки» → указать путь к папке с `Wow.exe`
4. «Добавить» → заполнить ник (или оставить пустым для записи-аккаунта), логин, пароль, класс, реалм, realmlist, при необходимости — секрет 2FA
5. Двойной клик по записи → автозаход

Конфиг хранится рядом с программой (`characters.json`), а если папка программы недоступна для записи — в `%LOCALAPPDATA%\АвтологинWOW\`. Пароли и 2FA-секреты по умолчанию зашифрованы DPAPI.

## Поддерживаемые серверы

Из коробки настроен на WoWCircle (`logon.wowcircle.com`). Список реалмов и realmlist-серверов редактируется в Настройках.

Технически работает с любым 3.3.5a-сервером, где клиент тот же `Wow.exe build 12340` и серверная сторона принимает стандартный поток автологина. Параметры передаются в Wow.exe через файл `autologin.json`, который патч `AwesomeWotlkLib.dll` читает при старте клиента (поля: `login`, `password`, `realmlist`, `realmname`, `character`, опционально `totp_secret`).

2FA должна реализовывать `TokenEnterDialog` GlueXML или вызов Lua `AcceptToken_AccountLogin`.

## Сборка из исходников

Нужен Python 3.10+, Visual Studio 2022 с Desktop development with C++ (для clang-cl) и [Inno Setup 6](https://jrsoftware.org/isdl.php).

```bat
build.bat
```

Это поставит зависимости (Nuitka, zstandard, pefile, pystray, Pillow), скомпилирует `launcher.py` в standalone-сборку и упакует её в установщик через Inno Setup. Готовый артефакт — `dist\installer\Manager_WOW.exe`. Первая сборка может занять 5-10 минут (Nuitka подкачает MinGW/clang при необходимости).

DLL `AwesomeWotlkLib.dll` уже собран и лежит в репозитории. Если хочешь пересобрать его сам — открой `awesome_wotlk_src/build_x86/Project.sln` в Visual Studio 2022.
